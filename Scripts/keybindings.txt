Lauch programs:
    MOD + SHIFT + ENTER             | terminal (alacritty)
    MOD + C                         | browser (firefox)
    MOD + CONTROL + C               | browser (chrome)
    MOD + F                         | file manager (ranger)
    MOD + CONTROL + F               | file manager (ranger) (toggle scratchpad)
    MOD + SHIFT + F                 | file manager (pcmanfm)
    PRTSCR                          | print screen (flameshot)
    MOD + G                         | open gtop (toggle scratchpad)
    MOD + CONTROL + G               | open htop (toggle scratchpad)
    MOD + CONTROL + D               | open todo list (toggle scratchpad)

Multimedia controls:
    MOD + P                         | open pulsemixer (toggle scratchpad)
    MOD + CONTROL + M               | mute
    MOD + CONTROL + U               | volume up 5%
    MOD + CONTROL + D               | volume down 5%

Dmenu scrips:
    MOD + X                         | dmenu menu
    MOD + R                         | dmenu_run
    MOD + CONTROL + R               | dmenu_run (gives sudo prompt to run programs in root)
    MOD + A                         | app switcher (dmenu)
    MOD + N                         | network manager (dmenu)
    MOD + B                         | bluetooth manager (dmenu)
    MOD + U                         | mount drives (dmenu)

Machine state controls:
    MOD + SHIFT + P                 | shutdown machine
    MOD + SHIFT + R                 | reboot machine
    MOD + SHIFT + S                 | slock
    MOD + SHIFT + Q                 | close dwm (log out)

Layouts:
    MOD + T                         | tiled view
    MOD + M                         | monocle (full screen) view
    MOD + Y                         | center view

Window controls:
    MOD + SHIFT + C                 | close window
    MOD + SHIFT + CONTROL + Q       | restart dwm
    MOD + SPACE                     | toggle floating for the current window
    MOD + ENTER                     | switch master window
    MOD + I                         | increase number of masters
    MOD + D                         | decrease number of masters
    MOD + SHIFT + T                 | toggle the bar
    MOD + [1-9]                     | jump to Nth tag
    MOD + TAB                       | jump to the previous tag
    MOD + [H or L]                  | move one tag in either direction
    MOD + SHIFT + [0-9]             | put the current window to Nth tag
    MOD + CONTROL + [1-9]           | add Nth tag to current view
    MOD + 0                         | show all tags
    MOD + SHIFT + 0                 | show all tags on the bar (Not only currently used tags)
    MOD + [J or K]                  | change focused window through the stack
    MOD + [{ or }]                  | expand window either left or right
    MOD + [Left or Right]           | change focused monitor

Gaps:
    MOD + ALT + 0                   | toggle gaps
    MOD + SHIFT + ALT + 0           | revert to default gaps
    MOD + ALT + [U or I or O]       | increase either all, inner, or outer gaps
    MOD + SHIFT + ALT [U or I or O] | decrease either all, inner, or outer gaps
    MOD + ALT + [H or V]            | increase horizontal or vertical outer gaps
    MOD + SHIFT + ALT + [H or V]    | decrease horizontal or vertical outer gaps

Help:
    MOD + SHIFT + ?                 | show this help
